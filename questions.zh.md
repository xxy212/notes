## Glassdoor 上的 Jane Street 问题

你会选择掷一个骰子并把结果平方，还是掷两个骰子并相乘？
  * `E[X^2] = 1/6(1+4+9+16+25+36)`，`E[XY] = E[X] E[Y] = 7/2*7/2`（独立性）。
  * 与 5 次掷骰中位数比较：`f(a) = 5C2 F(a)^2 f(a) (1-F(a))^2`。注意要在 `1-F(x)` 中包含较低数字。答案是 10.07，是 3 个选项中最低的。

数字乘积为 10000 的最小数。
  * 分解成因子 `22225555`，再合并并重排得到 `255558`。

给一个 `3x3x3` 立方体涂色，然后切成 `1x1x1` 小立方体。随机选一个小立方体并掷出，涂色面朝上的概率是多少？
  * 1 个中心块有 0 个涂色面，6 个面心块有 1 个，12 个棱块有 2 个，8 个角块有 3 个。
    `P(paint) = P(paint|ctr)P(ctr) + P(paint|mid)P(mid) + P(paint|edge)P(edge) + P(paint|cnr)P(cnr) = 1/3`。

50 以内奇数之和。
  * 总和为 `n(n+1)/2`，等于偶数和 + 奇数和。但每个偶数都只是对应奇数多 1。因此偶数和 + 奇数和 = `(奇数和 + n/2) + 奇数和`，解出奇数和。

A 和 B 打棒球两局。A 能否在两局中都有更高的打击率，但 B 的总体打击率更高？
  * 可以，A 分别为 `9/10` 和 `71/100`，B 分别为 `89/100` 和 `7/10`。

想连续赢 2 场：在与强者 A 和弱者 B 比赛时，选择 ABA 还是 BAB？
  * 分别计算 `P(WWW) + P(LWW) + P(WWL)`，比较 `2ab-aab` 与 `2ab-bba`。应选择 ABA。

从 `{1,2,3...,29,30}` 中找一个子集，使所有数字两两互质。
  * 平凡子集 `{x}` 存在。
  * 最大子集是所有素数。

两个骰子点数和为偶数的概率。
  * 不要考虑 2,3,...,11,12，而是考虑状态空间 EE、EO、OE、OO。EE 和 OO 给出偶数和，因此总概率为 50%。

## Brain Stellar

俄罗斯轮盘：1 颗子弹 vs. 2 颗子弹。
  * 1 颗子弹：`bxxxxx, xbxxxx, xxbxxx, xxxbxx, xxxxbx, xxxxxb`。概率为 `1/5` vs `1/6`。
  * 2 颗子弹：`bbxxxx, xbbxxx, xxbbxx, xxxbbx, xxxxbb, bxxxxb`。概率为 `1/4` vs `1/3`。

海盗分 100 枚金币。如果 50% 同意，方案通过。否则杀掉船长并重来。
  * 递归思考每个海盗在杀掉船长后的处境。
  * `n = 1: 100`
  * `n = 2: 100 0`
  * `n = 3: 99 0 1`
  * `n = 4: 99 0 1 0`
  * `n = 5: 98 0 1 0 1`

一只老虎可以吃羊，但吃完会变成羊。有 100 只老虎时，羊能活下来吗？
  * 递归思考每只老虎吃掉羊后的处境。
  * `n = 1: no` —— 单只老虎会吃。
  * `n = 2: yes` —— 老虎不会吃，因为吃完后会被另一只吃。
  * `n = 3: no` —— 老虎吃掉羊，然后是安全的……
  * `n = 100: yes` —— 偶数会导致羊不被吃。

为什么去掉相对角的棋盘无法用多米诺骨牌铺满？
  * 每块多米诺覆盖 1 个黑格和 1 个白格。相对角颜色相同，所以现在黑白数量不等。

哪个更大：`e^pi` 还是 `pi^e`？
  * 展开法：`exp(pi/e - 1) > 1 + (pi/e - 1)` --> `exp(pi/e)/e > pi/e` --> `exp(pi) > pi^e`，证毕。
  * `x^(1/x)` 方法（一般化）：`e^pi ~ pi^e` --> 两边取 `1/(pi e)` 次方 --> `e^(1/e) ~ pi^(1/pi)`。对 `log(x^(1/x))` 求导得 `(1 - ln(x))/x^2`。因此 `log(x^(1/x))` 的导数在 `x = e` 前严格递增、之后严格递减。由于 log 单调，所以 `e^(1/e) > pi^(1/pi)`。

可利用 `x^(1/x)` 在 `x = e` 处达到峰值来解：
  * `e^pi^2 ~ pi^e^2` --> 两边取 `1/(pi pi e e)` 次方再平方；
  * `e^sqrt(pi) ~ pi^sqrt(e)` --> 两边取 `1/sqrt(pi e)` 次方再开平方。

在任意骰子点数上下注 $1。掷 3 个有偏骰子。每出现一个下注点数赢 $1。
  * 期望收益：`E[X] = 3 p (1-p)^2 + 6 p^2 (1-p) + 3 p^2`，然后对所有 `p` 从 0 到 1 积分得到 `3/2`，再减去下注成本 $1。
  * 或者，对每个数字都下注，则保证赢 $3。每张票的期望收益是 $0.50。

从装有 50B 和 50R 的 urn 中取球。
  * 颜色变化的期望次数：`Xi` 表示第 i 个位置为 B 且第 i+1 个位置为 R。`P(Xi) = 50/100 50/99`，答案为 `1/2 50/99 * 99 * 2`（BR 和 RB 两种）= 50。

弓箭手射 2 箭，并被告知第 1 箭比第 2 箭好。
  * 第 3 箭最好的概率：列出 `123 132 213 231 312 321`，只保留 `123 132 312`，答案 = `1/3`。

从 `kB 1R` 的 urn 中取球。第一个拿到 R 的人获胜。
  * 先手还是后手：列出可能性。`RB BR`（无所谓），`RBB BRB BBR`（P1），`RBBB BRBB BBRB BBBR`（无所谓），`RBBBB BRBBB BBRBB BBBRB BBBBR`（P1）。所以 k 为偶数时无所谓，k 为奇数时选先手。

罐中有 `10R 20B 30G` 球。
  * 最后一个 R 后还剩 B 和 G 的概率：设为 `xxR1xxR2...xxR10xx`，找 `P(B 在 R 前耗尽) = 29C9/30C10 = 1/3`。`P(G 在 R 前耗尽) = 39C9/40C10 = 1/4`。交集 = `2 * 1/12 = 1/6`（2 表示顺序）。答案 = `1 - (1/4 + 1/3 - 1/6) = 7/12`。
  * 或者，考虑最后一颗糖是 B 的概率为 `20/60`。然后排列剩余 40 颗糖，其中最后一颗为 G 的概率为 `30/40`。同理最后一颗为 G。答案 = `20/60 30/40 + 30/60 20/30 = 7/12`。

面条问题：碗中有 `n=100` 根面条。每次拿一个端点并随机连接另一个端点，直到不能继续。
  * 用递归观察模式，从 `n=1` 开始。
  * 环数量期望：递推 `E[N] = 1/(2n-1) (E[N-1] + 1) + (2n-2)/(2n-1) E[N-1]`。`E[N] = 1 + 1/3 + 1/5 + ... + 1/(2N-1)`。`N=100` 时答案约为 3.28。
  * 一个大环的概率：递推 `Pn = 1 2/3 4/5 ... (2N-2)/(2N-1)`。`N=100` 时答案约为 8.8%。

有放回地从 `1,2...,n` 中抽 n 个数。
  * 不同值的期望数量：`D = n - F`，所以 `E[D] = n - E[F]`。`E[F] = sum(P(Xi)) = n P(Xi) = n (1-1/n)^n`，答案为 `n(1 - (1-1/n)^n) ~= n(1-1/e)`。

在圆上放 n 个点，然后画出连接所有点的凸包。
  * 凸包包围圆心的概率：这个事件等价于不存在一个半圆包含所有点。`P(0 semicircle) = 1 - P(>= 1 semicircle) = 1 - n (1/2)^(n-1)`，其中每个点都可作为半圆起点。构造通过该点和圆心的半圆。

不断加入 `U(0,1)` 变量，直到总和超过 1。
  * `P(N>=n)`：`P1 = 0`，`P2 = 1/2`（单位正方形中设三角形），`P3 = 1/6`（单位立方体中设金字塔）……`Pn = 1/n!`。
  * `E[N] = sum(n/n!) = e`。
  * 或者，将 `E[N]` 设为 `x` 的函数：`f(x) = 1 + integral from 0 to x (f(x-y))`。求导得到 `f'(x) = f(x)`，所以 `f(x) = k exp(x)`。`f(0) = 1` 给出 `k = 1`，因此 `f(1) = e`。

从 0 开始随机游走，直到碰到 `+a` 或 `-b`。
  * `P(先碰到 +a 而不是 -b)`：令 `n = a+b`。要求 `f(0)`，且知道 `f(a)=1`、`f(-b)=0`。反向代入可得 `f(-b+x) = x/(x+1) f(-b+x+1)`，所以 `f(a-1) = (n-1)/n`。回代得到 `f(-b+x) = x/n`。最终 `f(0) = f(-b+b) = b/n = b/(a+b)`。

## QuantNet

注意：像 MC 积分中的拒绝采样一样，如果超集是均匀采样的，那么子集也会被均匀采样。
  * 这是一个有用提醒：即使某些结果会提前结束，也要列出所有结果。

下注三局两胜比赛是 2 局结束还是 3 局结束？
  * `AAA, AAB, ABA, BAA, ABB, BAB, BBA, BBB` —— 50-50，但如果一队更强，则概率会向 2 局结束倾斜。
  * 列出所有 3 局选项。不要对某些情况在 2 局就停止！这样会因为局数少（熵更低）而低估。
  * 也可用二项分布：`P(goes to 3) = P(1W1L) = 2C1 p (1-p)`。

如果两队实力相当，世界大赛进入第 7 场的概率。
  * 画 Bertrand 投票问题图。需要先到 `(3,3)`，再到 `(4,3)` 或 `(3,4)`。到 `(3,3)` 的总方式数是 `6C3 = 20`，再乘以 2 得 40。所有 7 场顺序数为 `2^7 = 128`。最终答案为 `40/128`。
  * 也可用二项分布：`P(goes to 6) = P(3W3L) = 6C3 p^3 (1-p)^3`。

在正方形边上随机放 3 个点，落在不同边上的概率？
  * `4! / 4^4 = 6/64`。

10 个人握手总次数。
  * `10C2`。

前 2 张牌为红色则获胜。选 52 张牌还是 104 张牌？
  * 104 张牌的有限样本影响更小。

计算 `39 x 41`。
  * `(40-1)(40+1) = 1600 - 1 = 1599`。

匿名计算平均工资。
  * 第一个人写下自己的工资加一个随机数。之后每个人加上自己的工资。最后第一个人减去随机数得到总和。

`99^99` 有多少位？
  * `99^99 = (10*9.9)^99 = 10^99 * 9.9^99 = 10^99 * 10^(99 log10 9.9)`……然后我卡住了。
  * 用极限到 e：`99^99 = 100^99 * (1-1/100)^99 ~ 10^198 * 1/e`，所以有 198 位。
  * 它一定小于 `10^198`（因为小于 `100^99`）。现在设 `(100/99)^99 = (1+1/99)^99 < e < 10`。所以 `10 * 99^99 > 100^99 --> 99^99 < 10^197`。因此 `99^99` 位数夹在 198 和 199 之间，必为 198。

100 名乘客中的醉酒乘客问题。
  * 这不是随机排列问题！
  * 设基本情况：`n=2: AB or BA --> 1/2`，`n=3: ABC BAC BCA CAB --> 1/2`。
  * 一般地，`1/n` 概率坐正确座位，`1/n` 概率坐你的座位，`1/n` 概率挤走第 2 个人（然后他变成醉酒者），`1/n` 概率挤走第 3 个人……
  * `1/n (1 + (n-1 problem) + (n-2 problem) + ... + (2-problem) + 0) = 1/n (1 + (n-2)*1/2) = 1/2`，因为所有子问题解都是 `1/2`（归纳）。

1 到 100 的和。
  * 使用求和公式：`100*101/2 = 50*101 = 5050`。

用 3 加仑和 5 加仑水壶量出 4 加仑。
  * `(0,0) (3,0) (0,3) (3,3) (1,5) (1,0) (0,1) (3,1) (0,4)`。

A 有 `n+1` 枚硬币，B 有 `n` 枚。A 正面数严格更多的概率？
  * （来自 green book）对前 n 枚，`x + y + x = 1`，其中 `x` 为 A 正面更多的概率，`y` 为正面数相同的概率。`P(A wins) = 0.5*y + x = 0.5(1-2x) + x = 0.5`。
  * 也可用对称性论证。

不放回抽 2 张同色牌的概率。
  * 任意抽第一张颜色，第二张概率 `25/51`。

3 个灯开关，只允许访问房间 1 次。如何确定线路？
  * 先设置 `(on,off,off)` 一段时间。然后快速设置 `(off,on,off)` 并检查。热但关着的是第一个开关，亮着的是第二个，另一个关着的是第三个。

100 次抛硬币出现偶数个正面的概率？
  * 对奇数次抛掷，分布对称（例如 0H - 5H）。再增加一次抛掷不会显著改变，概率至少在 `n` 趋于无穷时趋于零。
  * 更简单：正面数奇偶只由最后一次抛掷决定。50-50。

3R、2B 中没有 BB 相邻的概率？
  * 总排列数为 `5C2 = 10`。含 BB 的排列数为 `4C1 = 4`。答案为 `6/10`。

只由 0 和 1 组成的最小 15 的倍数。
  * `15n = (10+5)n = 10n + 5n`。`n` 必须为偶数，因为所有 5 的奇数倍末位为 5。所以 `15(2n) = 30n = 10 * 3n`，它必须是 3 的倍数。`3,6,9,12,15,18,...,99,102,105,108,111`。可跳过中间，因为十位不是 0/1。答案 = `111 * 10 = 1110`。

1027 是素数吗？
  * 不是，因为 `1027 = (10^3 + 3^3) = (a^3 + b^3)`，可以因式分解。

波动率会提高看涨期权价格吗？
  * 会，距离到期时间更长也会提高价格。

盒子里放 2B 和 2R。每次猜对得 $1。游戏 EV 是多少？
  * 列出所有游戏：`BBRR, BRBR, BRRB, RRBB, RBRB, RBBR`。猜更可能的颜色，平局时猜 B。然后计算每种情况下的奖金。EV = `17/6`。

`2^230` 的个位数字是什么？
  * 2 的幂个位数模式为 `2,4,8,6,2,4,8,6...`。230 是偶数但不是 4 的倍数，所以答案是 4。

## Jane Street 指南

2 个 d6 骰子乘积为奇数的概率？3 个骰子呢？
  * 列表。2 个骰子为 `9/36 = 1/4`。扩展到三维后，每个奇数格概率再减半，得到 `1/8`。
  * 或者：乘积为奇数当且仅当每个数都是奇数。因此概率为 `(1/2)^n`。

已知 3 次抛硬币中至少有 1 个 T，出现 2 个 H 的概率？
  * 列出可能：`HHH, HHT, HTH, THH, TTH, THT, HTT, TTT`。去掉第一项，答案为 `3/7`。
  * 或者：`P(2H | >=1T) = P(2H and >=1T) / P(>=1T) = 3/7`。

连续出现 2 个 H 的概率/等待时间。
  * 建立马尔可夫链线性方程并求解。
  * 注意 `E[T] = E[start]`，因为抛到 T 就像重新开始。
  * 递推关系从终点开始：`E[nH] = 1/2(E[n-1H] + 1) + 1/2(E[n-1H] + 1 + E[nH])`。
    * 到达 `n-1H` 后，再抛 H（成功）或 T（失败并重新开始）。

## Aaron Cao 指南

编写代码生成给定数字的整数分拆。
  * 可用回溯生成分拆本身。
  * 可用 DP 生成分拆数量。与 P-518 Coin Change 2 相同。

如何按协方差聚类变量？
  * 简单：使用 k-means，成对距离设为 `1 - Corr[A,B]`。

洗牌后相邻 A 对数的期望？
  * 简单方法：把 2 张 A 粘在一起有 12 种方式，然后排列 51 张新牌。`12/52 = 3/13`。
  * 指示变量：第 i 和 i+1 张牌都是 A 的概率为 `4C2 / 52C2 = 6 / 1326`。然后 `E[sum(E[Xi])] = sum(E[Xi]) = 51 * 6/1326 = 3/13`。

多重继承会产生什么问题？
  * 菱形问题：D 实现 B 的虚函数还是 C 的虚函数？
  * C++ 需要显式声明，例如 `B::fct`。

生成给定自然数的所有分拆。

给定字典，生成一个词的所有有效变位词。
  * 暴力：生成所有变位词，然后检查哪些有效。耗时 `O(s!)`。可用回溯。
  * 更好：对字典中所有单词排序，或转成 set/Counter。检查排序后的词/Counter 是否在字典中。耗时 `O(w s)` 或 `O(w s log s)`。

抽牌直到抽到 A 的期望抽牌数？
  * 设置 `___A1___A2___A3___A4___`，每个空白期望为 `48/5`。加 1 得 `53/5 = 10.6`。

Alice 从 `U(0,1)` 采样一次。Bob 一直采样直到超过 Alice。求 `E[Bob samples]`。
  * `E[B] = integral(E[B|A=a] P(A=a) da) = integral(1/(1-x) dx)`，从 0 到 1，使用几何变量期望。积分不收敛。

给定 n 维空间中的 n 个单位向量，找一个与它们夹角都相同的向量。

求对角线上为 n、其他位置为 1 的矩阵的特征值。

X 和 Y 是 iid 标准正态。求 `P(X | X+Y >0)`。
  * 长方法：求 `P(X, X+Y) = [[1,0],[1,1]] [X, Y] = [X, X+Y]`，其 `mu = [0,0]`，`cov = [[1,1][1,2]]`。然后积分。

Bob 声称能通过味道辨别糖果颜色。他尝了 3 颗并说对 2 种颜色。总共有 5 种颜色。他靠谱吗？如果是 40/100 呢？

$100 本金、`p = 0.5`、盈利为 `1.2x` 的最优下注规模。

## 其他

计算 `1/(1 + 1/(1 + ...))`。
  * 递归：`x = 1/(1+x) --> x^2 - x + 1 = 0 --> x = (-1 +/- sqrt(5))/2`。

折棍问题。
  * 两段中较短段的期望长度：从 0 到 1/2 积分 `x`，再从 1/2 到 1 积分 `1-x`，等于 `2 * 1/8 = 1/4`。
  * 三段中最短段的期望长度：设单位正方形，指定 `X <= Y`（正方形左上 1/2），`X <= Y-X`，`X <= 1-Y`（可能最小值的 1/3）。从 `x=0` 到 `x=1/3`，对 `y=2x` 到 `y=(1-x)` 积分 `x`。再乘以 3（最小段可能性）和 2（`X <= Y`）。这是 pdf 的归一化。最终答案 `1/9`。
  * 能形成三角形的概率：设单位正方形，并指定 `X <= Y`、`X <= Y-X <= 1-Y`。这些条件需要归一化常数 `2 * 3! = 12`。区域由 `Y=1/2`、`Y=(X+1)/2` 和 `Y=2X` 限定。两个三角形面积为 `1/64 + 1/192`，乘以 12 得 `1/4`。
    * 三角形面积可用 `1/2 det(x0 y0 1, x1 y1 1, x2 y2 1)`。
    * 或者设等边三角形，点表示木棍长度（由高线定理）。如果点在中间 triforce 内则成功。

扔鸡蛋：给定 2 个鸡蛋，设计一个算法以最小化最坏情况下确定鸡蛋破裂楼层阈值的时间。
  * 最简单：1,3,5,...,99，然后立刻知道。最坏情况为 50。
  * 跳跃：10,20,30,...100，然后逐层测试 9 次。最坏情况为 19。
  * 动态：从 x 开始，然后递增 `(x-1),(x-2),...,1`。最坏情况为 x。解 `1+2+...+x = 100` 得 `x = 13.6`，所以答案是从 14 开始，然后 27，然后 39，然后 50……必要时逐层递增。

## PuzzledQuant

你有大小为 `2, 4, 8, ..., 2^n` 的砝码和一个两边天平。有多少种逐个放砝码的方法，使左边始终大于右边？
  * 听起来类似 Bertrand 投票问题。
  * 有 n 个砝码、2 个边，每次可选边，因此总放置方式为 `2n * 2(n-1) * ... = n! 2^n`。
  * 最大的砝码必须放在左边。
  * 类似 Catalan 数，考虑 `xxxXxxx`，其中 X 表示放置最大砝码。可以放成 `... xXxxx xxXxx xxxXx ...`。
  * 从 0 到 n-1 求和：`((n-1)Ci fi (n-1-i)! 2^(n-1-i))`，其中第一项表示选择并放置 i 个砝码，然后把最大砝码放在左边，再任意顺序放置剩余砝码。

有 n 对编号手套。随机配对后，使每一对编号差不超过 1 的概率？
  * 将每只标为 A 和 B。生成配对的总方式数为 `(2n)!/((2!)^n n!)`，去掉每对内部和配对之间的顺序。
  * 考虑 `1A`。可以是 `1A1B`、`1A2A` 或 `1A2B`。得到递推 `f(n) = f(n-1) + 2 f(n-2)`。
  * 最终答案由递推方程 `x^2 - x - 2 = 0` 和初始条件得出：`f(n) = 1/3 (2^(n+1) + (-1)^n)`。

单位球面上 z 坐标的方差。
  * 直接解法：对 `theta` 和 `phi` 积分 `z = cos(theta)`。
  * 巧妙解法：`E[X^2 + Y^2 + Z^2] = 1 --> E[Z^2] = 1/3 --> Var[Z] = 1/3`。
